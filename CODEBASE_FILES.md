# Complete Codebase File Guide

This file documents the use of every source/config/schema file in this repository, excluding `.git` internals.

## Root configuration
- `package.json`: NPM scripts and dependency manifest.
- `package-lock.json`: Locked dependency tree generated by npm.
- `vite.config.js`: Vite setup (React plugin + alias resolution).
- `tailwind.config.js`: Tailwind theme/content scanning configuration.
- `postcss.config.js`: PostCSS plugins used in build.
- `eslint.config.js`: ESLint flat configuration and project lint rules.
- `jsconfig.json`: JavaScript tooling config (path aliases/intellisense).
- `components.json`: shadcn UI generator configuration.
- `index.html`: HTML entry template and root mount element.
- `README.md`: Project setup/run guide.

## Public assets
- `public/vite.svg`: Static Vite logo asset.

## Data schemas (`entities/`)
- `entities/PlantDatabase.json`: Plant reference schema (traits, growing requirements, pests, companions, propagation, harvest windows).
- `entities/PlantDiagnosis.json`: Diagnosis record schema (plant/disease, severity, confidence, symptoms, treatment, status, notes).
- `entities/Task.json`: Farm task schema (type, status, due date, recurring rules, dependencies, assignees).
- `entities/WeatherLog.json`: Manual weather logging schema.
- `entities/Treatment.json`: Treatment library schema (ingredients, method, frequency, safety, rating).
- `entities/ForumPost.json`: Community post schema.
- `entities/ForumComment.json`: Community comment schema.
- `entities/CropPlan.json`: Crop planning schema (dates, milestones, schedules, status).
- `entities/DiagnosisFeedback.json`: Feedback schema for diagnosis quality/corrections.
- `entities/OutbreakReport.json`: Community outbreak reporting schema with geo fields.
- `entities/PestPrediction.json`: Predicted outbreak/risk schema and preventative guidance.

## App entry and layout
- `src/main.jsx`: Primary React entrypoint and route table.
- `src/App.jsx`: Alternate app router shell (kept for compatibility/reference).
- `src/Layout.jsx`: Active top-navigation + `<Outlet />` page wrapper.
- `src/globals.css`: Global styling/Tailwind layers.

## Utility and error helpers
- `src/lib/utils.js`: Shared `cn` utility (`clsx` + `tailwind-merge`).
- `src/lib/PageNotFound.jsx`: 404 page shown for unmatched routes.
